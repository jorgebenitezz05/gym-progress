# GymProgress

GymProgress es una aplicación full stack para registrar entrenamientos, rutinas, ejercicios y progreso físico.

El proyecto está dividido en dos partes:

- Backend desarrollado con Java y Spring Boot.
- Frontend móvil desarrollado con Flutter.

## Objetivo

Crear una aplicación real que permita a un usuario:

- Registrarse e iniciar sesión.
- Crear ejercicios.
- Crear rutinas.
- Registrar entrenamientos.
- Guardar series, repeticiones y peso.
- Consultar historial.
- Ver estadísticas de progreso.

## Tecnologías previstas

### Backend

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Spring Security
- JWT
- PostgreSQL
- Swagger
- Docker

### Frontend

- Flutter
- Dart
- HTTP/Dio
- Material Design
